Password

Your password to allow access to the event creation and update features of the Network Startup Resource Center Network Education calendar web site.

Enter the password twice, to ensure that it has beeen typed accurately.

Notes:

  1. The password must be at least six(6) characters long, and must contain at least one number or element of punctuation.
  2. Be certain to remember or record your password.
  3. If you forget your password you can request that it be sent via email to you.
  4. You can change your password using the on-line user update form.
