URL

If your materials are available on the web at a specific URL please enter this URL in here. In some cases this can be very useful, particulary if someone who is considering downloading your files wishes to look at them first.

If there is a URL that points to a workshop site that lists your files among others, then feel free to use this URL as well. URLs can be of the form:

http://your.url.here/filename.ext

your.url.here/filename.ext

ftp://your.url.here/directory/filename.ext

Any valid URL format is acceptable. The result should be that if someone clicks on the URL that you enter, then their web browser will correctly open a page to your materials.
