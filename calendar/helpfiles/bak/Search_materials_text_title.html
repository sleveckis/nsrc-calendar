Title

If you choose to search on 'Title' then if the actual title of the materials that have been submitted contain the string you are searching for these material items will be displayed in the search results.

When someone submits a set of materials (a set can be a single file) they give their materials a descriptive title. This is not a concise or reliable way to find, for instance, every item concerning 'Network Security' as the original submitter may not title a set of materials with the text 'Network Security' even if the topic of the materials is related.

You may wish to include Category Names as part of your text string search as well.
