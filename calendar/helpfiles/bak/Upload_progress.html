Progress Bar

If you check this box, then a popup window will appear with a progress bar that indicates the total size of all your files chosen to upload, total amount uploaded, estimated upload time, time left, and estimated upload speed.

As the PHP scripting language has no way (as of version 4) of displaying a progress bar like this the solution uses a Javascript to call a CGI (Perl) file. If your Browser has Javascript disabled, you can uncheck this box and a straight PHP-method for doing file uploads will be used instead.

Note: if you uncheck the progress bar box, then your screen will clear and there will be no feedback during the file upload process. Once your file or files upload, the screen will refesh with the next step and you can continue.

Final note: If you close the progress bar window your file upload will still be active. In order to stop a file upload you must press the stop button in your main Browser window.
